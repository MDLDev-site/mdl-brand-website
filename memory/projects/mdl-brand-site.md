# MDL Brand Site — Project Context

## Overview
Public-facing marketing and brand site for MDL. Upcoming project.

## Tech Stack (Planned)
- **Framework:** Astro
- **Architecture:** Islands architecture — zero JS by default, hydrate only interactive components
- **Rendering:** Static Site Generation (SSG)
- **Deployment:** Vercel
- **Content:** Astro Content Collections for structured data

## Architecture (Planned)
- Static pages with selective hydration
- Content collections for blog posts, team bios, feature showcases
- MDL design system tokens (Inter font, #2F34FE brand blue)
- SEO-first: pre-rendered pages, structured data, sitemap
